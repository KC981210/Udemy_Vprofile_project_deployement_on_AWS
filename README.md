# Udemy_Vprofile_project_deployement_on_AWS
This repository contains project files and implementation details from a guided Udemy course, “Decoding DevOps – From Basics to Advanced Projects with AI” by Imran Teli. The project demonstrates a step-by-step deployment of a web application stack—Tomcat, RabbitMQ, Memcached, and MySQL—using AWS Infrastructure as a Service (IaaS). It focuses on practical DevOps concepts, cloud infrastructure provisioning, and application deployment workflows on AWS.
